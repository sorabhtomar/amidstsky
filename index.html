Amidst Sky

A free, premium quality, responsive WordPress theme with a minimal layout and beautiful typography. Amidst Sky is designed for personal blogs and magazines with long-form content.

Download

Amidst Sky is a premium quality, free WordPress 4+ theme created by Deepak Bansal. It can be yours right now, all you need to do is download the theme from WordPress respository. You can use this theme for any purpose, personal or commercial.

This customizable WordPress theme uses Genericons along with Google Fonts Libre Baskerville and Montserrat. With this theme, just the slightest splash of color and custom header and/or background can make a huge impact on the overall presentation and design.

Amidst Sky is available under GNU General Public License v2 or later.

Download from GitHub

Customize

Once you've downloaded and activated the theme you can customize the theme from 'Theme Customizer' feature in Appearance menu. Amidst Sky by default uses a light color scheme. But you can customize it however you want. Amidst Sky theme is very flexible and you can even create a dark shade website just by modifying few colors.

WordPress core features like Custom Header images and Background images are also supported and you can use them to improve the over all look and feel of your website.
